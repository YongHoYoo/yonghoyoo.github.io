SMT Inspection Equipment Path Optimization

Research on minimizing inspection time for SMT (Surface-Mount Technology) equipment

Problem Overview

SMT inspection equipment optically inspects objects (pads, components, etc.) placed on a PCB. The system uses multiple Fields of View (FOVs), and every component within each FOV must be inspected.

  • Minimize the number of FOVs – Avoid redundant imaging.
  • Optimize the inspection path – Reduce total inspection time.

Mathematical Formulation

  • FOV Selection → Set Cover Problem: Each FOV covers multiple components. Goal: smallest set of FOVs that covers all components.
  • Path Optimization → TSP Variant:
    • Standard TSP: Visit all nodes and return to the start.
    • Variant: Fixed start, free end (no return required).

Approach

  • Heuristic methods: Greedy, 2-opt, Simulated Annealing.
  • Learning-based methods:
    • Transformer model for path prediction.
    • Beam Search & Monte Carlo Tree Search (MCTS) for improved sequence generation.
  • Simulation & Evaluation: Python, NumPy, Pandas, Plotly.
  • Performance comparison between heuristic and Transformer-based approaches.

What Made This Problem Special

Unlike a standard TSP, this was a complex constraint optimization problem:

  • Probe movement speed and acceleration limits.
  • FOV-specific imaging time (image count & FPS).
  • 3D reconstruction and inspection time per component.
  • Multi-threaded inspection with thread allocation limits.
  • Memory constraints during execution.

This required combining route optimization, scheduling, and hardware-aware constraints into one integrated solution.

Results

  • Significant reduction in inspection time.
  • Reduced number of required FOVs.
  • Improved productivity without hardware changes.
  • Clear performance gains over traditional heuristics.
